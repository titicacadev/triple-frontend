# useSessionCallback

sessionId가 있는 환경에서만 주어진 콜백을 실행하는 함수를 반환하는 훅.
sessionId가 없으면 로그인 유도 모달을 띄웁니다.

## Usage

```tsx
const callback = useSessionCallback()
```
