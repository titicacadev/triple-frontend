# useClientApp

User Agent가 앱 환경일 때만 주어진 콜백을 실행하는 함수를 반환하는 훅.
앱이 아닐 때는 transition 모달을 띄웁니다.

## Usage

```tsx
const clientApp = useClientAppCallback(TransitionType.General, callback)
```
