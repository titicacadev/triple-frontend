# Text

Text 를 표현할때 사용하는 컴포넌트입니다.

<Preview>
  <Story name="Basic">
    <Text size="tiny" color="gray" center={false}>
      호텔 최저가 보상!
    </Text>
  </Story>
</Preview>

```jsx
<Text size="medium" color="blue" onClick={() => 'clicked'}>
  호텔 최저가 보상!
</Text>
```

```js
size?: GlobalSizes | number

whiteSpace?: CSS.WhiteSpaceProperty
center?: boolean
textAlign?: CSS.TextAlignProperty
underline?: boolean
inline?: boolean
inlineBlock?: boolean
margin?: MarginPadding
padding?: MarginPadding
ellipsis?: boolean
maxLines?: number
strikethrough?: boolean
```

<Text size="medium"> medium: 15px </Text>

<Text size="big"> big: 19px </Text>
<Text size="huge"> huge: 21px </Text>
<Text size="massive"> massive: 24px </Text>
<Text size={50}>size={50}</Text>

## TextAlign

`center` 는 deprecated property 입니다. (`textAlign` 이 없을때만 동작합니다.)

<Text center margin={{ bottom: 10 }} style={{ border: '1px solid gray' }}>
  center
</Text>
<Text
  center
  textAlign="right"
  margin={{ bottom: 10 }}
  style={{ border: '1px solid gray' }}
>
  center and textAlign right
</Text>
<Text
  textAlign="center"
  margin={{ bottom: 10 }}
  style={{ border: '1px solid gray' }}
>
  textAlign center
</Text>
<Text center style={{ border: '1px solid gray' }}>
  center
  <Text textAlign="left" style={{ border: '1px solid lightgray' }}>
    textAlign left
  </Text>
</Text>

```jsx
<Text center>center</Text>
<Text center textAlign="right">center textAlign="right"</Text>
<Text textAlign="center">textAlign="center"</Text>
<Text center>
  center
  <Text textAlign="left">textAlign="left"</Text>
</Text>
```
