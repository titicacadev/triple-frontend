# EventMetadatProvider

## Usage

```tsx
export function Page({ children }) {
  return (
    <EventMetadataProvider
      eventMetadataContext={{
        userId: 'userId',
      }}
    >
      {children}
    </EventMetadataProvider>
  )
}
```

### eventMetadataContext
