## Input

기본 Input 입니다.

```js
id?: string
type?: string
value?: string
error?: string
placeholder?: string
mask?: string
maskChar?: string | null
onChange?: (e: React.SyntheticEvent, value: string) => any
onBlur?: (e: React.FocusEvent<any>) => any
```

### Input - Status

Focus 와 Error 두 가지의 상태 값을 가지고 있습니다.
focused, error props 를 이용하여 상태를 다룰 수 있습니다.

<Input
  placeholder="이름을 입력해주세요"
  help="고객님의 요청사항은 해당 호텔에 전달됩니다만 호텔 사정에 따라 필요하신 내용이 이루어지지 않을 수 있으니 많은 양해 바랍니다."
  error
/>

### Input - Mask

mask props 를 이용하여 mask 를 추가 할 수 있습니다.

  <Story name="Input - Mask">
    <Input mask="99/99" maskChar={null} value={1230} />
  </Story>
</Preview>

## Select

option 을 선택 할 수 있는 SelectBox 입니다.

```js
id?: string
name?: string
value?: any
placeholder?: string
options?: SelectOption[]
focused?: string
error?: string
onBlur?: (e: React.FocusEvent<any>) => any
onChange?: (e?: React.SyntheticEvent, value?: any) => any
```

## Confirm Selector

```js

value: any
placeholder: string
onChange?: (e?: React.SyntheticEvent, value?: any) => any

borderless?: boolean
fillType?: FillType ('full' | 'border' | 'text')
children?: React.ReactNode
padding?: MarginPadding
error?: string
```
