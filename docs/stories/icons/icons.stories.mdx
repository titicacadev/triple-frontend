## Theme

Icon 은 기본적으로 `color` prop 이 최우선의 아이콘 컬러를 적용받습니다.

두번째로 ThemeProvider 에 지정된 `colors.primary` 값을 설정하게 됩니다.

그리고 마지막으로는 기본 블랙 컬러인 `gray` 값을 적용받게 됩니다.

<Preview>
  <Story name="Theme">
    <ThemeProvider theme={{colors: {primary:Colors.blue}}}>
      <>
        <Back strokeWidth={1.6} />
        <Back strokeWidth={1.6} color={Colors.mint} />
      </>
    </ThemeProvider>
  </Story>
  <Story name="Out of theme context">
    <Back strokeWidth={1.5} />
  </Story>
</Preview>
