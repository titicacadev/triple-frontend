# Theme Provider 

Theme Provider 로 감싸진 styled component 의 내부에는 theme set 이 props 으로 내려갑니다.    
기본 theme 는 light theme 를 사용합니다

```js
<ThemeProvider>
  <Text> Theme Provider 로 감싸진 텍스트 </Text>
</ThemeProvider>

// text.ts

const Text = steyld.div`
  ${({ theme: { colors } }) => `
    color: ${colors ? colors.gray : default Color}
  `}
`
```

## Dark Mode

isDark 를 이용하여 Theme 의 전환이 가능합니다.

```js
const [isDark] = useState(true)

<ThemeProvider isDark={isDark}>
  Dark Mode 
</ThemeProvider>
```

## Custom Theme 

기본 Light, Dark 외에도 커스텀 테마를 적용 할 수 있습니다.

```js
const customTheme = {
  colors: {
    gray: '블라블라'
  }
}

<ThemeProvider theme={customTheme}>
  Dark Mode 
</ThemeProvider>
```
