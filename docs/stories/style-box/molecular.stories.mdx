# Style Box

Style Box 는 컴포넌트를 작성할때 필요한 다양한 스타일을 지원하여 빠르게 컴포넌트를 만들 수 있도록 도와줍니다.

<br />

## [border-radius](https://developer.mozilla.org/ko/docs/Web/CSS/border-radius)

border-radius CSS 속성은 요소의 바깥 테두리 경계의 모서리를 둥글게 만듭니다.

```
borderRadius (value?: number)

${borderRadius()} or ${borderRadius(10)}
```

## [Centered](#centered)

Block 요소를 가운데 정렬합니다.

```
centered (value?: boolean)

${centered()} or ${centered(true)}
```

## [Clearing](#clearing)

float 된 요소들을 clear 시킵니다

```
clearing (value?: booelan)

${clearing()} or ${clearing(true)}
```

## [Ellipsis](#ellipsis)

글자가 고정된 너비보다 넘칠 경우 말줄임 처리합니다.

```
ellipsis (value?: booelan)

${ellipsis()} or ${ellipsis(true)}
```

## [Horizontal Scroll](#horizontal-scroll)

너비가 넘칠 경우 가로스크롤을 만들어줍니다.

```
horizontalScroll (value?: booelan)

${horizontalScroll()} or ${horizontalScroll(true)}
```
