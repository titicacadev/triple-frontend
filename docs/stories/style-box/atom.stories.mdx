# Style Box

Style Box 는 컴포넌트를 작성할때 필요한 다양한 스타일을 지원하여 빠르게 컴포넌트를 만들 수 있도록 도와줍니다.

<br />

## [Position](https://developer.mozilla.org/ko/docs/Web/CSS/position)

CSS position 속성은 문서 상에 요소를 배치하는 방법을 지정합니다.top, right, bottom, left 속성이 요소를 배치할 최종 위치를 결정합니다.

`default unit 은 px 입니다.`

```
position (value?: CSS.PositionProperty | Position)

${position('relative')} or ${position({ type: 'absolute', top: 20, right: 30})}
```

## [Box-Sizing](https://developer.mozilla.org/ko/docs/Web/CSS/box-sizing)

box-sizing CSS 속성은 요소의 너비와 높이를 계산하는 방법을 지정합니다.

```
boxSizing (value?: CSS.BoxSizingProperty)

${boxSizing()} or ${boxSizing('')}
```

## [Clear](https://developer.mozilla.org/ko/docs/Web/CSS/clear)

clear CSS 속성은 요소가 선행 부동(floating) 요소 다음일 수 있는지 또는 그 아래로 내려가(해제되어(cleared))야 하는 지를 지정합니다.

```
clear (value?: CSS.ClearProperty)

${clear()} or ${clear('both')}
```

## [Float](https://developer.mozilla.org/ko/docs/Web/CSS/float)

CSS 속성(property) float 은 한 요소(element)가 보통 흐름(normal flow)으로부터 빠져 텍스트 및 인라인(inline) 요소가 그 주위를 감싸는 자기 컨테이너의 좌우측을 따라 배치되어야 함을 지정합니다.

```
float (value?: CSS.FloatProperty)

${float()} or ${float('left')}
```

## [Height](https://developer.mozilla.org/ko/docs/Web/CSS/height)

height CSS 속성은 요소의 높이를 지정합니다.

`default unit 은 px 입니다.`

```
height (value?: number)

${height()} or ${height(100)}
```

```
${minHeight()} or ${minHeight(100)}
```

## [Margin](https://developer.mozilla.org/ko/docs/Web/CSS/margin)

margin CSS 속성은 요소의 네 방향 바깥 여백 영역을 설정합니다.

`default unit 은 px 입니다.`

```
margin (value?: MarginPadding)

${margin()} or ${margin({ top: 10, right: 10, bottom: 10, left: 10 })}
```

## [Padding](https://developer.mozilla.org/ko/docs/Web/CSS/padding)

padding CSS 속성은 요소의 네 방향 안쪽 여백 영역을 설정합니다.

`default unit 은 px 입니다.`

```
${padding()} or ${padding({ top: 10, right: 10, bottom: 10, left: 10 })}
```

## [Text-Align](https://developer.mozilla.org/ko/docs/Web/CSS/text-align)

CSS text-align 속성은 블록 요소나 테이블 칸 상자의 가로 정렬을 설정합니다.

```
textAlign (value?: CSS.TextAlignProperty)

${textAlign()} or ${textAlign('center')}
```

## [User-Select](https://developer.mozilla.org/ko/docs/Web/CSS/User-Select)

user-select는 사용자가 텍스트를 선택할 수 있는지 여부를 결정하는 속성입니다.

```
userSelect (value?: CSS.UserSelectProperty)

${userSelect()} or ${userSelect('none')}
```

## [White-Space](https://developer.mozilla.org/ko/docs/Web/CSS/White-Space)

white-space 속성은 요소가 공백 문자를 처리하는 법을 지정합니다.

```
whiteSpace (value?: CSS.WhiteSpaceProperty)

${whiteSpace()} or ${whiteSpace('pre-line')}
```

## [Display](https://developer.mozilla.org/ko/docs/Web/CSS/Display)

display CSS 속성은 요소를 블록과 인라인 요소 중 어느 쪽으로 처리할지와 함께, 플로우, 그리드, 플렉스처럼 자식 요소를 배치할 때 사용할 레이아웃을 설정합니다.

```
display (value?: CSS.DisplayProperty)

${display()} or ${display('inline-block')}
```
